# Skills Test Specification

Specification document for testing Agent Skills behavior, output contracts, and routing.

This repository is a Starlight documentation site.

The production site is published under:

```text
https://wangwei1237.github.io/Skills-Test-Specification/
```

## Development

```bash
npm install
npm run dev
```

With the configured Astro `base`, the local entry point is:

```text
http://localhost:4321/Skills-Test-Specification/
```

## Build

```bash
npm run build
```

Main content lives in `src/content/docs/*.mdx`.
