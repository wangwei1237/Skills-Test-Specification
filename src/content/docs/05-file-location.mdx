---
title: "文件位置"
description: "Skills Test Specification - 文件位置"
---

每个 Skill 至少维护一份测试文件：

```text
${ABS_ROOT}/skills/<skill-name>/tests/skill_cases.yaml
```

推荐结构：

```bash
${ABS_ROOT}/
└── skills/
    └── <skill-name>/
        ├── SKILL.md
        ├── scripts/
        │   └── script.py
        └── tests/
            └── skill_cases.yaml
```
